Auxiliary Data

This web service provides auxiliary data written by the telescope sensors during data taking. You can select the name of the table and the time range for which you want the information. Simply point your browser to

/aux/<NAME>&from=XXXXX&to=xxxxx

and you'll recieve the data encoded in .json format.

The following link for example will show you some weather information from timestamp 2013-02-01-00:40UTC to 2013-02-02-23:10UTC /aux/TNG_WEATHER_DATA?from=2013-02-01-00:40UTC&to=2013-02-02-23:10UTC

For a full list of tables/service endpoints visit /services
